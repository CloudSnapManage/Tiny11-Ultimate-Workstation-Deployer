The Architecture of Automation

Setting up a new Windows environment is usually a day-long headache of downloads, reboots, and path configurations. This script transforms that experience into a single-command deployment.

1. Hardware-Aware Intelligence

Unlike generic batch files, this script queries WMI and CIM to identify your specific components. It detects the 13th Gen Intel i7 and RTX 4050 to ensure drivers are applied correctly and performance modes like HAGS (Hardware Accelerated GPU Scheduling) are toggled on.

2. Size-Optimized Multi-Tier Install

Designed for efficiency on 5MB/s connections, the script uses a tiered approach:

üöÄ Tiers 1-2

Small utilities like 7-Zip, Git, and VLC. Instant wins to get your system feeling "ready" fast.

üõ† Tiers 3-5

Heavyweights like Android Studio, Anaconda, and NVIDIA drivers. Scheduled last to avoid bottlenecking.

3. Professional Environment Mapping

The script doesn't just install apps; it builds a workspace. It creates dedicated data zones with desktop shortcuts:

  • C:\Dev Source Code
  • C:\AI_Models LLM & AI Weights
  • C:\Media Video/Photo Assets
  • C:\Games Game Library
