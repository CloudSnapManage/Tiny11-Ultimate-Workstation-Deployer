Manual setup is the enemy of productivity. This project was born from the need to move from a fresh Tiny11 install to a full-stack MERN and local AI development environment in under 60 minutes.

Quick Start Deployment

Copy and paste the following command into an Administrative PowerShell terminal:

Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://raw.githubusercontent.com/CloudSnapManage/Tiny11-Ultimate-Workstation-Deployer/main/Ultimate_Setup.ps1'))

The "Single Folder" Philosophy

The script maintains OS stability by keeping binaries in Program Files while siloing your actual work into high-speed data zones with automatic desktop shortcuts:

  • C:\Dev: MERN & Mobile projects
  • C:\AI_Models: Local LLMs and Diffusers
  • C:\Media: Creative assets and exports
